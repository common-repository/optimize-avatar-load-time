Optimize your page performance by loading avatar only a single time ( in reviews, comments, etc. ) from gravatar.com and store it locally.

== Description ==

= Introduction =

Store and Load Gravatar From Local - makes your site faster by loading avatar only first time from gravatar.com. This plugin stores the avatar image locally when the page is loaded for the first time. Then after the image will be picked from local.

<blockquote>

<ul>
<li>Loads the gravatar first time page loads and saves it in local.</li>
<li>From then onwards the avatar loads from local.</li>
<li>A default image will be loaded if there is no avatar found.</li>
<li>Improved PageSpeed, YSlow performance scores. Check it using GTmetrix.</li>
</ul>
